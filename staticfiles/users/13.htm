13.1 Data Mining негізгі кітапханалары

Data Mining жүзеге асырудың бірден бір программаларының бірі Python программалау ортасы. Ал, Python программалау ортасында үлкен көлемді деректерге амалдарды қолдану үшін ең алдымен:

-       scipy;

-       numpy;

-       matlibplot;

-       pandas;

-       sklearn кітапханаларын орнатып алуымыз керек.

SciPy пакеттік бумасын (соның ішінде NumPy) орнатудың және пайда-ланудың ең жақсы тәсілі-Anaconda дистрибутивін орнату. Бұл дистрибуцияны орнату арқылы сіз бірнеше жүздеген әртүрлі пакеттерді орнатасыз. Бұл шама-дан тыс болып көрінуі мүмкін, бірақ іс жүзінде бұл өте ыңғайлы болып шығады - сізге қажет нәрсенің бәрі сіздің қолыңызда. Сондай-ақ, Jupiter веб-орамасын (обертка) орамасында интерактивті Python оболочкасын аласыз. Мұндай ортада жұмыс істеу өте ыңғайлы. Мысалы, төмендегі 12-ші суретте өлшемі 10х10 кездейсоқ сандардың матрицасын құру және SymPy кітапханасын қолдана отырып, қабылдау үшін ыңғайлы нәтиже қалай көрінетіндігі берілген. [39].

13.3 Pandas кітапханасының мүмкіндіктері

Pandas - тің негізгі міндеті-деректерді талдау әлеміне енгендердің барлы-ғына барлық құрылыс блоктарын ұсыну [41].

Pandas кітапханасын орнатудың қарапайым әдісі - жиналған шешімді пайдалану, яғни Anaconda немесе Enthought арқылы орнату.

Сонымен қатар, Pandas-ты келесі команданы пайдаланып PyPI көмегімен орнатуға болады:

 

Егер сіз Linux дистрибутивінде жұмыс жасасаңыз және осы шешімдерді пайдаланбауды шешсеңіз, онда pandas-ты кез-келген басқа пакет сияқты орнатуға болады.

Ол үшін Debian және Ubuntu-да команда қолданылады:

sudo apt-get install python-pandas

Ал OpenSuse және Fedora үшін келесі командалар тізбегін пайдаланамыз:

zypper in python-pandas

Егер сіз Windows-та жұмыс жасасаңыз және пакеттерді әрдайым соңғы нұсқасы орнатылатындай етіп басқарғыңыз келсе, онда Windows үшін модульдерді әрқашан жүктеуге болатын ресурс бар: Christoph Gohlke's Python Windows-қа арналған кеңейтілген пакеттер. Әр модуль 32 және 64 биттік үшін WHL(wheel) форматында келетін жүйелер. Орнату үшін pip қолданбасын пайдалану керек:

pip install SomePackage-1.0.whl

 

Мысалы, pandas орнату үшін келесі пакетті табу және жүктеу қажет:

 

pip install pandas-0.22.0-cp36-cp36m-win_amd64.whl

 

Модульді таңдағанда Python нұсқасы мен архитектурасын таңдау маңызды. Сонымен қатар, егер NumPy пакеттері қажет болмаса, онда pandas тәуелділіктерге ие болады.

Бұл тәсілдің кемшілігі-сізге қажетті нұсқаларды және әртүрлі пакеттер арасындағы тәуелділікті таңдауға көмектесетін менеджерсіз пакеттерді бөлек орнату керек. Сонымен қатар, модульдерге ыңғайлы болу және тарату таңдағанына қарамастан, соңғы нұсқаларды алу мүмкіндігі бар.

Scikit-learn-бұл Data Science және Machine Learning үшін кеңінен қолданылатын Python пакеттерінің бірі. Ол көптеген операцияларды орындауға мүмкіндік береді және көптеген алгоритмдерді ұсынады. Scikit-learn сонымен қатар өзінің класстары, әдістері мен функциялары туралы керемет құжатта-маны, сонымен қатар қолданылатын алгоритмдердің сипаттамасын ұсынады [42].

Scikit-Learn:

-  деректерді алдын-ала өңдеу;

-  өлшемді азайту;

-  модельді таңдау;

-  регрессия;

-  жіктеу;

-  кластерлік талдауды қолдайды.

Сондай-ақ, ол сіздің модельдеріңізді тексеру үшін пайдалануға болатын бірнеше мәліметтер жиынтығын ұсынады.

Scikit-learn машиналық оқытуға байланысты барлық нәрсені жүзеге асырмайды. Мысалы, ол үшін жан-жақты қолдау жоқ:

-  нейрондық желілер;

-  өзін-өзі ұйымдастыратын карталар (Кохонен желілері);

-  ассоциативтік ережелерді оқыту;

-  арматурамен оқыту (reinforcement learning).

Scikit-learn NumPy және SciPy-ге негізделген, сондықтан Scikit-learn-ді тиімді пайдалану үшін осы екі кітапхананың негіздерін түсіну қажет.

Scikit-learn-бұл ашық бастапқы пакет.

Scikit-learn мүмкіндіктерін қарастырайық.

Сіз өзіңіздің деректеріңізді машинаны оқыту алгоритмдеріне дайындау үшін scikit-learn қолдана аласыз: деректерді стандарттау немесе қалыпқа келтіру, категориялық айнымалыларды кодтау және басқалар.

Өлшемді азайту көп өлшемді деректер жиынтығының маңызды компо-ненттерін таңдауды немесе алуды қамтиды. Scikit-learn өлшемді азайтудың бірнеше тәсілдерін ұсынады. Олардың бірі-негізгі компоненттерді талдау (PCA).

Машиналық оқыту модельдерін оқыту және тестілеу үшін сіз кездейсоқ мәліметтерді ішкі жиындарға бөлуіңіз керек. Бұған кірістер де, олардың тиісті шығулары да кіреді. Sklearn функциясы.model_selection.train_test_split() мұндай жағдайларда пайдалы.

Деректер жиынтығын әдеттегі бөлуден басқа, scikit-learn кросс-тексеруді жүзеге асыру, тор іздеу арқылы модельдеріңіздің гиперпараметрлерін орнату, модельдің жұмысын көрсететін көптеген шамаларды есептеу құралдарын ұсынады (мысалы, анықтау коэффициенті, RMS қатесі, түсіндірмелі ауытқу көрсеткіші, қате матрицасы, жіктеу туралы есеп, f-көрсеткіштер және тағы басқалар). Scikit-learn сіздің модельдеріңізді зерттеуге және тексеруге жарамды бірнеше мәліметтер жиынтығын ұсынады. Негізінен, бұл белгілі деректер жиынтығы. Олар модельдерді тестілеу үшін жеткілікті мөлшерде деректерді ұсынады, сонымен бірге өте үлкен емес, бұл оқытудың қолайлы ұзақтығын қамтамасыз етеді.

Scikit-learn сызықтық регрессиядан бастап, көпмүшелік регрессия, тірек векторларының регрессиясы, шешім қабылдау ағаштары және т.б. арқылы жақын көршілердің k әдісі, random forest және градиентті күшейту сияқты әдістер тобына дейін әртүрлі регрессия әдістерін қолдайды. Ол сонымен қатар нейрондық желілерді қолдайды, бірақ TensorFlow сияқты мамандандырылған кітапханалар сияқты емес.

Scikit-learn жіктеуді регрессия сияқты орындайды. Ол логистикалық регрессия және K-жақын көршілер, тірек векторлық әдіс, Байес классифи-каторы, шешім ағашы, сондай-ақ random forest, AdaBoost және градиентті күшейту сияқты жіктеу әдістерін қолдайды .

Кластерлеу - бұл scikit-learn-да кеңінен қолдау көрсетілетін бақылан-байтын оқыту саласы. K-орташа әдісіне қосымша жақындықты тарату әдісін, спектрлік кластерлеуді, агломерациялық кластерлеуді және т. б. қолдану мүмкіндігі бар.

Django-бұл Python-ның жоғары деңгейлі веб-инфрақұрылымы, ол қауіпсіз және қызмет көрсетуге ыңғайлы веб-сайттарды тез дамытуға мүмкіндік береді [43].

Веб-сайттарды жасау кезінде әрдайым ұқсас компоненттер қажет: пайдаланушыларды аутентификациялау әдісі (кіру, шығу, тіркеу), сайтты бас-қару тақтасы, пішіндер, файлдарды жүктеу құралдары және т. б. Осы мүмкін-діктердің барлығын Django көмегімен жүзеге асыруға болады.

Django-ның даму ортасы - жергілікті компьютерге орнату және оны Django қосымшаларын өндірістік ортаға орналастырмас бұрын әзірлеу және сынау үшін пайдалануға болады.

Django-ның өзі ұсынатын негізгі құралдар - бұл Django жобаларын құруға және олармен жұмыс істеуге арналған Python сценарийлерінің жиынтығы, сонымен қатар Django веб-қосымшаларын компьютердің веб-шолғышында жергілікті тестілеу үшін пайдалануға болатын қарапайым веб-сервер.

Даму ортасының бөлігі болып табылатын басқа да перифериялық құрал-дар бар. Оларға мәтіндік редактор немесе кодты өңдеуге арналған IDE және кодтың әртүрлі нұсқаларын қауіпсіз басқаруға арналған git сияқты бастапқы кодты басқару құралы кіреді.

Django ортасын орнату әдісі мен орны жағынан өте икемді. Django-ң мынадай жағдайлары болуы мүмкін:

-  түрлі операциялық жүйелерде орнатылған;

-  бастапқы кодтан, Python Package Index (Php) және көптеген жағдайларда кез-келген пакет менеджерінен орнатылған;

-  бөлек орнатылуы және конфигурациялануы керек әртүрлі дерекқорларды пайдалануға теңшелген;

-  негізгі Python қоршаған орта жүйесінде немесе бөлек Python виртуалды ортасында іске қосылу мүмкіндігі бар.

Django 4 негізгі дерекқорды қолдайды (PostgreSQL, MySQL, Oracle және SQLite), сонымен қатар басқа SQL және NoSQL дерекқорларына әртүрлі қолдау деңгейлерін беретін қоғамдық кітапханалар бар. Екі орта үшін біркелкі МҚ қолданған жөн.

Виртуалды ортаны құру үшін біз virtualenvwrapper (Linux және macOS X) және virtualenvwrapper-win (Windows) кітапханаларын қолданамыз. Олар өз кезегінде virtualenv құралын пайдаланады. Орауыш(обертка) құралы барлық платформаларда интерфейстерді басқаруға арналған үйлесімді интерфейсті ұсынады.

Windows ортасына Django-ны орнату жағдайын қарастырайық.

Енді сіз mkvirtualenv командасын қолдана отырып, жаңа виртуалды орта жасай аласыз. Команданы іске қосу кезінде сіз виртуалды ортаны орнатуды көре аласыз. Командалар тізбегін орындағаннан кейін жаңа виртуалды орта іске қосылады (Сурет 15).

 

 

Сурет 15- Виртуалды ортаның іске қосылуы

 

Django - ортасында пайдаланылатын пайдалы командалар тізбегі мынадай:

-  deactivate-Қазіргі Python виртуалды ортасынан шығу

-  workon-қол жетімді виртуалды орталардың тізімі

-  workon name_of_environment-нақты Python виртуалды ортасын белсендіру

-  rmvirtualenv name_of_environment-нақты виртуалды ортаны жою.

Виртуалды ортаны құрып, оған кіру үшін work on командасын шақыр-ғаннан кейін сіз Django орнату үшін pop3 қолдана аласыз. Ол үшін төмендегі команда тізбегі қолданылады:

pip3 install django

 

Сіз Django қондырғысын келесі команданы орындау арқылы тексере аласыз.
